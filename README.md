# Xsd Test
#### Just a tiny project for test adidas template XSD file validate.

- **XmlToGeneric**   
Convert XML to Model (better way).
- **XmlToObj**   
Convert XML to Object.
- **ObjToXml**   
Convert Object/Model to XML string.
- **XmlValidate**   
Using xsd file to vaildate XML format.
